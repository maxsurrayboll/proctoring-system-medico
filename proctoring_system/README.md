# Sistema de Monitoreo de Exámenes Médicos

Aplicación web desarrollada en Flask para supervisar evaluaciones académicas en el área de la salud.

## Funcionalidades

- Registro de usuarios
- Inicio de sesión seguro
- Panel de control
- Monitoreo con cámara
- Prevención de fraude académico

## Tecnologías

- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
